API 文档

下面列出了本服务提供的所有接口。

GET /

  • 功能：返回首页（index.html）

GET /docs

  • 功能：返回文档地址

GET /ping

  • 功能：健康检查

返回示例：

{
  "message": "pong"
}

GET /s2twp?text=…

  • 功能：将简体字转换为台湾正体（常用词汇）
  • 参数：
    • text（string，必填）：要转换的文本

返回示例：

{
  "text": "轉換後的繁體文字"
}

POST /s2twp

  • 功能：将简体字转换为台湾正体（常用词汇）
  • 参数：
    • text（string，必填）：要转换的文本

返回示例：

{
  "text": "轉換後的繁體文字"
}

POST /convert

  • 功能：根据 type 参数进行各种 OpenCC 转换

请求示例：

{
  "text": "要转换的文本",
  "type": "s2t"  // 转换类型，参考 /types 接口
}

返回示例：

{
  "text": "转换后的文本"
}

返回示例：
